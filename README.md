# SportApi

Необходимый софт:

- Node.js 10
- Docker

Локально базу данных можно запустить с помощью команды `docker-compose up -d`

Спецификация доступна по адресу `http://localhost:3000/api/`

```bash
$ npm i -g yarn
$ yarn install
```

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
